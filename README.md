## Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/Mayank-Golchha/Excel-MCP.git
   cd Excel-MCP
   ```

2. Windows
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt

   Mac/Linux
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt

3. Run the Server
    uv run --with pandas excel_mcp_server.py
